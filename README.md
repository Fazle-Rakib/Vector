#Vector
SWE 223 Game Project
Vector is a 2D platformer game. Player has to save himself from collision with different objects.
 A player will have three lives to complete each of 2 levels. Vector has nice background music 
and animation, which will create a nice gaming environment. 
*Language : Java
* Frameworks : libgdx

Note : It’s a project for SWE-224 Course (Object Oriented Programming Language Lab) at SUST.
